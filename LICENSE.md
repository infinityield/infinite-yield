## 🐛 Common Issues and Fixes

### ❌ Script doesn't execute / nothing happens
**Fix:** Make sure your executor is properly attached to Roblox. Re-inject the executor, then try executing the script again.

### ❌ Command bar doesn't appear on screen
**Fix:** Type `;cmds` in the chat, or check if your screen resolution is hiding the GUI. Try resetting your character first.

### ❌ Kill All command doesn't work
**Fix:** Some games have teleport or damage protection. The **kill all** feature may be blocked server-side. Try `;kill [playername]` on individual targets instead.

### ❌ Mobile executor crashes after pasting script
**Fix:** Ensure you're using the latest version of your mobile executor. Arceus X and Delta are most stable for the **infinite yield android script**.

### ❌ "attempt to index nil" Lua error
**Fix:** This usually means Roblox updated and broke a service reference. Check this repository for the latest patch version.

### ❌ Script loads but commands don't respond
**Fix:** Type `;prefix` to check your active command prefix. It may have been changed from the default `;`.

### ❌ Anti-cheat kicks player immediately
**Fix:** This game likely has server-side detection. Avoid using the script in that title. Stick to games without active anti-cheat.
